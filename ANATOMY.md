# Anatomy

This repo is an evidence-first writing harness.

- `.agent/`: product-neutral doctrine, capability registry, checklists, and templates.
- `.claude/`: Claude Code adapter surface generated from capability semantics.
- `.agents/`: Codex adapter surface generated from capability semantics.
- `state/`: live writing control plane.
- `paper/`: LaTeX source. Not the source of truth for facts.
- `lab/`: claim, evidence, result, citation, and lightweight generation ledgers.
- `release/`: tex-only export surfaces.
- `exemplars/`: rhetorical move maps only.
- `human/`: human-agent briefs, reviews, decisions, and inbox.
- `memory/`: current status, handoffs, worktree notes, and change control.
- `scripts/`: deterministic checks, exporters, and sync helpers.
